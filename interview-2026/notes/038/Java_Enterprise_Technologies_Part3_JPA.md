## Key Concepts Summary

### Entity Mapping
```
@Entity: Marks class as entity
@Table: Custom table name
@Id: Primary key
@GeneratedValue: ID generation
@Column: Column mapping
@Temporal: Date/time mapping
@Lob: Large object mapping
```

### Relationships
```
@OneToOne: 1:1 relationship
@OneToMany: 1:N relationship
@ManyToOne: N:1 relationship
@ManyToMany: N:M relationship
@JoinColumn: Foreign key column
mappedBy: Inverse side
```

### Inheritance
```
SINGLE_TABLE: One table with discriminator
JOINED: Parent + child tables
TABLE_PER_CLASS: Separate tables
@DiscriminatorColumn: Type column
@DiscriminatorValue: Type value
```

### Criteria API
```
CriteriaBuilder: Factory
CriteriaQuery: Query definition
Root: Entity root
Predicate: Where conditions
TypedQuery: Executable query
Metamodel: Type-safe attributes
```
