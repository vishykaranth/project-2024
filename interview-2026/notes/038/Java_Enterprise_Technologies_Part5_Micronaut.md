### Compile-Time Processing
```

Bean Definitions: Pre-computed metadata
AOP Proxies: Generated at compile time

```

```
Build-Time DI: Pre-computed dependency graph
Bean Scopes: @Singleton, @Prototype, @RequestScope
Factory Methods: @Factory, @Bean
Qualifiers: @Named
```

```
@Controller: REST endpoints
@Get, @Post, @Put, @Delete: HTTP methods
@PathVariable: Path parameters
@Body: Request body
@Produces, @Consumes: Content negotiation
```

### AOP
```
@Transactional: Transaction management
@Retryable: Retry logic
@Cacheable: Caching
@Validated: Validation
Compile-time proxy generation
```
