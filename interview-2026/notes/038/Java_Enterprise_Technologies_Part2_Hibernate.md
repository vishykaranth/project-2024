## Key Concepts Summary

### Entity Relationships
```
One-to-One: @OneToOne
One-to-Many: @OneToMany, @ManyToOne
Many-to-Many: @ManyToMany, @JoinTable
Cascade: CascadeType.ALL, PERSIST, MERGE, etc.
Fetch: FetchType.EAGER, LAZY
```

### Caching
```
First Level: Per session, automatic
Second Level: Shared, configurable
Query Cache: Cached query results
Strategies: READ_ONLY, READ_WRITE, NONSTRICT, TRANSACTIONAL
```

### Query Optimization
```
HQL: Object-oriented queries
Criteria API: Type-safe queries
Native SQL: Direct SQL
JOIN FETCH: Eager loading
Batch Fetching: Reduce N+1
```

### Transactions
```
Transaction: session.beginTransaction()
Flush: session.flush()
Commit: transaction.commit()
Rollback: transaction.rollback()
Isolation: Database isolation levels
```
