## Key Concepts Summary

### Data Warehouse Schemas
```
Star Schema:
- Denormalized dimensions
- Simple joins
- Fast queries
- Easy to understand

Snowflake Schema:
- Normalized dimensions
- Complex joins
- Less storage
- Better for hierarchies
```

### OLAP Operations
```
Slice: Select one dimension value
Dice: Select multiple dimension values
Roll-up: Aggregate to higher level
Drill-down: Go to detailed level
Pivot: Rotate cube view
```

### OLAP Types
```
MOLAP: Pre-aggregated, fast, requires storage
ROLAP: SQL-based, scalable, no pre-aggregation
HOLAP: Hybrid approach, best of both
```
