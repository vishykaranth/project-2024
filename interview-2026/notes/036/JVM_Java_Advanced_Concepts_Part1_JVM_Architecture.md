### JVM Architecture Components
```
1. Class Loader Subsystem
   - Loading, Linking, Initialization
   - Bootstrap, Extension, Application ClassLoaders
   - Delegation model

2. Runtime Data Areas
   - Method Area (shared, class metadata)
   - Heap (shared, objects)
   - Stack (per thread, method calls)
   - PC Register (per thread, instruction pointer)
   - Native Method Stack (per thread, JNI)

3. Execution Engine
   - Interpreter (fast start)
   - JIT Compiler (hot methods)
   - Garbage Collector (memory management)

4. Native Method Interface (JNI)
   - Bridge to native libraries
   - C/C++ integration
```
