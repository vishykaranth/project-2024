## Key Concepts Summary

### Virtual Threads
```
1. Virtual Threads
   - Lightweight threads (M:N mapping)
   - Millions of threads possible
   - Low memory overhead
   - Efficient blocking I/O

2. Creation
   - Thread.ofVirtual()
   - Executors.newVirtualThreadPerTaskExecutor()
   - StructuredTaskScope

3. Structured Concurrency
   - Hierarchical task structure
   - Automatic cancellation
   - Error propagation
   - Prevents leaks

4. Use Cases
   - High-concurrency I/O
   - Thread-per-request
   - Parallel processing
   - Not for CPU-intensive tasks
```
