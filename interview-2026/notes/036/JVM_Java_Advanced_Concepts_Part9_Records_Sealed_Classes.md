```

   - Compact constructors
   - Custom methods allowed

   - Restricted inheritance
   - Explicit permitted subclasses
   - Exhaustive pattern matching
   - Type safety

   - instanceof pattern matching
   - Switch expressions
   - Record patterns
   - Exhaustive checking
```
