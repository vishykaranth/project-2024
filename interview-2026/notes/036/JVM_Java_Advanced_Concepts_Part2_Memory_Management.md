## Key Concepts Summary

### Memory Areas
```
1. Method Area / Metaspace
   - Class metadata
   - Static variables
   - Runtime constant pool
   - Shared across threads

2. Heap
   - Young Generation (Eden, Survivor S0/S1)
   - Old Generation
   - Object storage
   - Shared across threads

3. Stack
   - Per thread
   - Method frames
   - Local variables
   - Operand stack

4. PC Register
   - Per thread
   - Instruction pointer

5. Native Method Stack
   - Per thread
   - JNI calls
```

### Memory Flags
```
-Xms: Initial heap size
-Xmx: Maximum heap size
-Xss: Stack size per thread
-XX:MetaspaceSize: Initial metaspace size
-XX:MaxMetaspaceSize: Maximum metaspace size
-XX:NewRatio: Old/Young ratio
-XX:SurvivorRatio: Eden/Survivor ratio
```
