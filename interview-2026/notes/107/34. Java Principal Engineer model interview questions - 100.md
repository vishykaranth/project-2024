# ☕ Java Principal Engineer — 100 Model Interview Questions

---

## I. JVM Internals & Performance (1–15)

1. How does the JVM decide when to do a Full GC vs Young GC?
2. Explain **GC pauses vs allocation rate** — which do you optimize first?
3. When would you choose G1 over ZGC (or vice versa)?
4. What JVM metrics tell you **memory pressure before OOM**?
5. Explain **safepoints** and why they matter.
6. How does **object allocation** actually work in the JVM?
7. What causes **GC thrashing** in production?
8. Explain **TLABs** and why they help.
9. How do you debug **high CPU but low throughput** in Java?
10. What’s the real cost of **boxing/unboxing** at scale?
11. How does **JIT compilation** change performance over time?
12. What JVM flags do you *never* tune blindly?
13. How do you reason about **heap vs off-heap** memory?
14. What happens during a **Stop-The-World** pause exactly?
15. When does GC tuning make things worse?

---

## II. Concurrency & Multithreading (16–30)

16. Explain the **Java Memory Model** in practical terms.
17. What guarantees does `volatile` give — and what doesn’t it?
18. When do `synchronized` blocks outperform `ReentrantLock`?
19. How does `CompletableFuture` fail in production?
20. Explain **thread starvation vs deadlock**.
21. What’s wrong with unbounded thread pools?
22. How do you size thread pools for IO vs CPU workloads?
23. Explain **false sharing** and how to detect it.
24. When do atomics become a bottleneck?
25. What concurrency bugs *don’t* show up in tests?
26. How does backpressure work in Java systems?
27. What happens when interrupts are ignored?
28. Explain **happens-before** using a real bug.
29. Why are async stacks hard to debug?
30. When does lock-free code hurt performance?

## III. Collections, Data Structures & Memory (31–40)

31. Why does `HashMap` resize cause latency spikes?
32. How does `ConcurrentHashMap` avoid global locks?
33. When does `CopyOnWriteArrayList` make sense?
34. Explain memory impact of **object graphs**.
35. Why do caches cause GC pressure?
36. What’s the real cost of `LinkedList`?
37. How do weak/soft references behave under GC pressure?
38. What happens when collection cardinality explodes?
39. How do you detect memory leaks without heap dumps?
40. Why do LRU caches fail under load?

43. What causes socket exhaustion?
44. Why does serialization become a bottleneck?
45. Protobuf vs JSON vs Avro — real tradeoffs?
46. How does connection pooling fail?
47. Explain **head-of-line blocking**.
48. What’s wrong with default HTTP client configs?
49. How do retries amplify network failures?
50. When does async IO not help?

54. What breaks when you add async inside a transaction?
55. How does Spring Boot auto-config hurt startups?
56. When does dependency injection become a liability?
57. How do you debug classpath hell?
58. Why do context refreshes cause outages?
59. What’s the real cost of AOP?
60. How do frameworks complicate observability?

* **Tradeoff articulation**
* **Operational instincts**
* **Teaching clarity**

If you answer **30–40 of these well**, you pass.
If you answer **10 with real outages**, you’re hired.
