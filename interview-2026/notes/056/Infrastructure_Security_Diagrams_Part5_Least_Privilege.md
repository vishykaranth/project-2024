## Key Concepts Summary

### Least Privilege
```
Principle: Minimum necessary access
Benefits: Reduced attack surface
Implementation: RBAC, ABAC, JIT
Monitoring: Regular access reviews
```

### RBAC
```
Roles: Collections of permissions
Users: Assigned to roles
Hierarchical: Role inheritance
Centralized: Easy management
```

### Zero Trust
```
Verify: Always authenticate
Least Privilege: Minimum access
Assume Breach: Continuous monitoring
Never Trust: Explicit verification
```

### JIT Access
```
On-Demand: Request when needed
Time-Limited: Auto-expire
Approval: Workflow required
Audit: Full logging
```
