## Key Concepts Summary

### Defense in Depth Principles
```
Multiple Layers: Redundant protection
Diversity: Different security controls
Fail-Safe: Default deny
Monitoring: Continuous surveillance
Response: Incident handling
```

### Network Defense
```
Perimeter: Firewalls, DDoS protection
Segmentation: Network isolation
Monitoring: IDS/IPS
Encryption: Data in transit
```

### Application Defense
```
WAF: Web application firewall
Secure Coding: Input validation
RASP: Runtime protection
Testing: Security testing
```

### Data Defense
```
Encryption: At rest, in transit, in use
Access Control: RBAC, encryption keys
Backup: Encrypted backups
Monitoring: Data access logs
```
