## Overview

This video introduces a qualification system to filter requests and protect your time, ensuring you only engage with valuable opportunities while politely declining time-wasters.

## The Qualification System

### The Framework

```
┌─────────────────────────────────────────────────────────┐
│         Developer Qualification System                 │
└─────────────────────────────────────────────────────────┘

For Every Request, Ask:

1. Is this aligned with my goals?
2. Does this create value?
3. Is this the best use of my time?
4. Can someone else handle this?
5. Is this a learning opportunity?

If 3+ Yes: Engage
If 2 or fewer: Decline or Delegate
```

## Qualification Questions

### Question 1: Goal Alignment

**Ask**: Does this align with my career goals?

**Evaluate**:
- Advances my skills?
- Builds visibility?
- Creates impact?
- Opens opportunities?

**Decision**:
- ✅ Yes → Consider engaging
- ❌ No → Likely decline

### Question 2: Value Creation

**Ask**: Does this create measurable value?

**Evaluate**:
- Solves real problem?
- Has business impact?
- Improves systems?
- Advances projects?

**Decision**:
- ✅ High value → Engage
- ❌ Low value → Decline

### Question 3: Time Efficiency

**Ask**: Is this the best use of my time right now?

**Evaluate**:
- Higher priority than current work?
- Can wait?
- Urgent and important?
- Better alternatives?

**Decision**:
- ✅ Best use → Engage
- ❌ Not optimal → Decline or defer

### Question 4: Delegation Potential

**Ask**: Can someone else handle this effectively?

**Evaluate**:
- Within others' expertise?
- Learning opportunity for them?
- Not my core responsibility?
- Others available?

**Decision**:
- ✅ Can delegate → Refer to others
- ❌ Needs me → Consider engaging

### Question 5: Learning Opportunity

**Ask**: Is this a valuable learning opportunity?

**Evaluate**:
- New skills to learn?
- Exposure to new areas?
- Career-advancing knowledge?
- Worth the time investment?

**Decision**:
- ✅ Valuable learning → Engage
- ❌ No learning → Likely decline

## Application Examples

### Example 1: Meeting Invitation

**Request**: "Can you join this planning meeting?"

**Qualification**:
1. Goal alignment? ❌ Not related to my goals
2. Value creation? ❌ No clear value
3. Time efficiency? ❌ Not best use of time
4. Can delegate? ✅ [Colleague] can represent
5. Learning opportunity? ❌ Nothing new

**Decision**: Decline
**Response**: "[Colleague] has been working on this and can provide the technical perspective. I'm happy to review the decisions afterward."

### Example 2: Technical Question

**Request**: "How do I configure this tool?"

**Qualification**:
1. Goal alignment? ❌ Not advancing my goals
2. Value creation? ❌ Low value
3. Time efficiency? ❌ Not optimal
4. Can delegate? ✅ Documentation exists
5. Learning opportunity? ❌ Nothing new

**Decision**: Refer to documentation
**Response**: "The configuration is documented here: [link]. If you have specific questions after reviewing, I'm available during office hours (Tuesdays 2-3 PM)."

### Example 3: High-Value Project

**Request**: "Can you lead this architecture redesign?"

**Qualification**:
1. Goal alignment? ✅ Advances architecture skills
2. Value creation? ✅ High business impact
3. Time efficiency? ✅ Best use of time
4. Can delegate? ❌ Needs my expertise
5. Learning opportunity? ✅ Valuable experience

**Decision**: Engage
**Response**: "Yes, I'd be interested in leading this. Let me understand the requirements and propose an approach."

## Response Templates

### Template 1: Decline with Referral

**Structure**:
"I appreciate you thinking of me. [Person/Resource] is better suited for this because [reason]. I'm happy to [supporting action] if helpful."

**Example**:
"Thanks for reaching out. [Colleague] has been working on similar features and would be perfect for this. I can provide context or review the approach if that helps."

### Template 2: Defer to Office Hours

**Structure**:
"I'm focused on [priority] right now. I'm available for [type of help] during my office hours [time]. For urgent issues, [alternative]."

**Example**:
"I'm in deep work on [project] right now. I'm available for technical questions during my office hours (Tuesdays 2-3 PM), or you can Slack me and I'll respond within 2 hours."

### Template 3: Request More Information

**Structure**:
"I'd like to help, but I need to understand [specifics] first. Can you provide [information] so I can evaluate if this is the best use of my time?"

**Example**:
"I'm interested, but I need to understand the scope and timeline. Can you share the requirements and deadline so I can assess if I can commit the time this deserves?"
