## Key Optimization Strategies

### Summary
```
1. Base Image:
   - Use Alpine or Distroless
   - Avoid full OS images

2. Multi-Stage Builds:
   - Separate build and runtime
   - Copy only artifacts

3. Layer Caching:
   - Order: Stable â†’ Changing
   - Copy dependencies before code

4. Dependencies:
   - Production only in final image
   - Use npm ci --only=production

5. .dockerignore:
   - Exclude node_modules, .git, logs
   - Reduce build context

6. Layer Minimization:
   - Combine RUN commands
   - Clean up in same layer
```
