```
- Max attempts
- Retryable exceptions
- Non-retryable exceptions
- Timeout handling

```

### Best Practices
```
1. Retry only transient failures
2. Use exponential backoff
3. Add jitter to prevent synchronization
4. Set reasonable max attempts
5. Implement fallback mechanisms
6. Monitor retry metrics
7. Don't retry on permanent failures
```
