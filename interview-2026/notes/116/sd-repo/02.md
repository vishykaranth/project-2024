This repository is not a collection of notes.

It is a **long-term, evolving knowledge graph for mastering System Design** —  
from fundamentals to Staff+ / Principal level thinking.

The structure follows a **reverse-tree model**:

```

Real-world systems
↓
Architecture patterns
↓
Distributed system building blocks
↓
Data & consistency models
↓
Scalability mathematics
↓
First principles

```

You can start at the top (interview mode)  
or at the bottom (deep fundamentals).
