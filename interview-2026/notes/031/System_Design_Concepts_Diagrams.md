## Diagram Drawing Tips

### 1. Use Standard Symbols
- **Boxes**: Components/Services
- **Arrows**: Data flow/Direction
- **Diamonds**: Decision points
- **Circles**: Start/End points

### 2. Label Everything
- Component names
- Data flow direction
- Protocol/Technology used
- Data types

### 3. Show Relationships
- Dependencies
- Communication patterns
- Data flow
- Control flow

### 4. Keep It Simple
- Focus on key components
- Avoid too much detail
- Use consistent notation
- Group related components
