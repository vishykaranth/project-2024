GraphQL is a query language and runtime for APIs.
It allows clients to request exactly the data they need.

Key Features:
1. Single Endpoint: One endpoint for all queries
2. Client-Driven: Clients specify what data they need
3. Strongly Typed: Schema defines all types
4. Introspection: Self-documenting API
5. Real-time: Subscriptions for live data
