Serverless is a cloud computing model where the cloud provider
manages the server infrastructure and automatically allocates
resources as needed.

Key Characteristics:
1. No Server Management: Provider handles infrastructure
2. Automatic Scaling: Scales based on demand
3. Pay-per-Use: Pay only for execution time
4. Event-Driven: Functions triggered by events
5. Stateless: Functions don't maintain state
6. Short-Lived: Functions execute and terminate
