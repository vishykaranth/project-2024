## 1. Photoelectric Effect

### Photoelectric Effect Setup
```
┌─────────────────────────────────────────────────────────────┐
│              Photoelectric Effect                           │
└─────────────────────────────────────────────────────────────┘

    Light (hν)
    │
    │
    ▼
┌──────────┐
│ Cathode  │───► e⁻ (photoelectrons)
│  (K)    │
└──────────┘
    │
    │ V (voltage)
    │
┌──────────┐
│  Anode   │
│   (A)    │
└──────────┘

Einstein's Equation:
hν = φ + K_max
hν = Work function + Maximum KE

Stopping Potential:
eV₀ = K_max = hν - φ
```

### Photoelectric Characteristics
```
┌─────────────────────────────────────────────────────────────┐
│              Current vs Voltage                              │
└─────────────────────────────────────────────────────────────┘

Current (I)
    │
    │───────────────
    │
    │
    │
    └──────────────► Voltage (V)
        -V₀
    
Saturation current: All photoelectrons collected
Stopping potential V₀: No current flows
```

## 2. Atomic Structure

### Bohr's Model
```
┌─────────────────────────────────────────────────────────────┐
│              Bohr's Atomic Model                             │
└─────────────────────────────────────────────────────────────┘

    n=4 ────────────
    n=3 ────────
    n=2 ────
    n=1 ──
    
    • (nucleus)
    
Energy levels:
E_n = -13.6/n² eV

Radius:
r_n = n²a₀
a₀ = 0.529 Å (Bohr radius)

Angular momentum:
L = nh/(2π) = nℏ
```

### Hydrogen Spectrum
```
┌─────────────────────────────────────────────────────────────┐
│              Hydrogen Spectrum Series                        │
└─────────────────────────────────────────────────────────────┘

    n=∞
    │
    │ Lyman (UV)
    n=1 ────────────
    │
    │ Balmer (Visible)
    n=2 ────────
    │
    │ Paschen (IR)
    n=3 ────
    │
    │ Brackett (IR)
    n=4 ──
    
Rydberg Formula:
1/λ = R(1/n₁² - 1/n₂²)
R = 1.097 × 10⁷ m⁻¹
```

## 3. Nuclear Physics

### Nuclear Reactions
```
┌─────────────────────────────────────────────────────────────┐
│              Nuclear Reactions                              │
└─────────────────────────────────────────────────────────────┘

Alpha Decay:
²³⁸U → ²³⁴Th + ⁴He
A → (A-4) + 4
Z → (Z-2) + 2

Beta Decay:
²³⁴Th → ²³⁴Pa + e⁻ + ν̄
Z → (Z+1)
A unchanged

Gamma Decay:
*²³⁴Th → ²³⁴Th + γ
No change in A or Z
```

### Binding Energy
```
┌─────────────────────────────────────────────────────────────┐
│              Binding Energy Curve                            │
└─────────────────────────────────────────────────────────────┘

BE per nucleon (MeV)
    │
    │     ╱╲
    │    ╱  ╲
    │   ╱    ╲
    │  ╱      ╲
    │ ╱        ╲
    │╱          ╲───────
    └──────────────────► Mass number (A)
    
Binding Energy:
BE = [Zm_p + (A-Z)m_n - m_nucleus]c²

Most stable: Fe-56 (peak)
Fusion: Light nuclei (left of peak)
Fission: Heavy nuclei (right of peak)
```

## 4. Radioactivity

### Radioactive Decay
```
┌─────────────────────────────────────────────────────────────┐
│              Radioactive Decay                              │
└─────────────────────────────────────────────────────────────┘

Activity (N)
    │
    │  ╱╲
    │ ╱  ╲
    │╱    ╲
    │      ╲
    │       ╲
    │        ╲
    └──────────────► Time
    
N = N₀e^(-λt)
A = A₀e^(-λt) = λN

Half-life:
T₁/₂ = ln(2)/λ = 0.693/λ

Mean life:
τ = 1/λ = T₁/₂/ln(2)
```

## 5. X-Rays

### X-Ray Production
```
┌─────────────────────────────────────────────────────────────┐
│              X-Ray Tube                                     │
└─────────────────────────────────────────────────────────────┘

    Cathode ────► e⁻ ────► Target (Anode)
    │                        │
    │                        │
    │                        ▼

λ_min = hc/(eV)

Characteristic X-rays:
Moseley's Law: √ν = a(Z - b)
```
