## 7. Kubernetes Implementation

### Ambassador Pod YAML
```yaml
apiVersion: v1
kind: Pod
metadata:
  name: app-with-ambassador
spec:
  containers:
  # Ambassador Container
  - name: ambassador
    image: envoyproxy/envoy:latest
    ports:
    - containerPort: 80
      name: http
    - containerPort: 443
      name: https
    env:
    - name: BACKEND_HOST
      value: "localhost"
    - name: BACKEND_PORT
      value: "8080"
    # Ambassador proxies requests to main app
  
  # Main Application Container
  - name: app
    image: myapp:latest
    ports:
    - containerPort: 8080
      name: app
    # App only communicates via ambassador
```

   - Use ConfigMaps/Secrets

   - Log all routing decisions
   - Metrics for routing
   - Distributed tracing

   - TLS termination
   - Authentication
   - Rate limiting
   - Input validation

6. Performance
   - Connection pooling
   - Caching
   - Load balancing
   - Timeout management

   - Retry logic
   - Circuit breakers
   - Timeout handling
   - Failover mechanisms
