## 3.1 Entry Point (Remote Facade)

```java
@RestController
@RequestMapping("/orders")
class OrderController {

    private final OrderService orderService;

    @PostMapping
    public OrderResponse placeOrder(@RequestBody PlaceOrderRequest request) {
        return orderService.placeOrder(request);
    }
}
```

### Pattern

* **PoEAA: Remote Facade**
* Thin controller
* No logic

---

## 3.2 Service Layer (Orchestration)

```java
@Service
public class OrderService {

    private final InventoryService inventory;
    private final PaymentService payment;
    private final OrderRepository orderRepo;
    private final EventPublisher events;

    @Transactional
    public OrderResponse placeOrder(PlaceOrderRequest req) {

        inventory.reserve(req.productId(), req.qty());

        payment.charge(req.paymentDetails());

        Order order = Order.create(req);
        orderRepo.save(order);

        events.publish(new OrderPlacedEvent(order.id()));

        return OrderResponse.success(order.id());
    }
}
```

### Why this maps to HLD

* Stateless
* Coordinates, doesn’t compute
* Single transaction boundary

---

### 🎯 LLD Kill Shot

> **“The service layer coordinates workflows; it doesn’t own business state.”**

## 3.3 Domain Model vs Transaction Script (Explicit Choice)

### Order Aggregate

```java
@Entity
class Order {

    @Id
    private String id;

    private OrderStatus status;

    static Order create(PlaceOrderRequest req) {
        Order o = new Order();
        o.status = CREATED;
        return o;
    }

    void markPaid() {
        if (status != CREATED) throw new IllegalStateException();
        status = PAID;
    }
}
```

### Why this fits

* Encapsulates invariants
* Matches HLD consistency rules

## 3.4 Inventory — Concurrency Control

```java
@Entity
class Inventory {

    @Id
    private String productId;

    @Version
    private long version;

    private int available;
}
```

```java
public void reserve(String productId, int qty) {
    Inventory inv = repo.findById(productId);
    if (inv.available < qty) throw new OutOfStock();
    inv.available -= qty;
}
```

### Pattern

* **Optimistic Locking**
* Matches “strong consistency” from HLD
